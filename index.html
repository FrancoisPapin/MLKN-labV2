📚 About MLKN-lab

MLKN-lab is a hierarchical knowledge network designed to bridge the gaps between scientific disciplines. Built from 326,861 rows of data, it maps 6 core domains, 25 academic disciplines, and 235 subdisciplines with over 320,000 connections.

This project aims to:

  • 🔗 Reveal hidden connections between disciplines (e.g., how Cognitive Psychology links to AI).
  • 📊 Quantify interdisciplinarity using network metrics.
  • 🌍 Support meta-research by providing a framework for studying the structure of science.
  • 🤖 Enhance AI systems by modeling human-like knowledge organization.

📌 Key Features

🔹 Hierarchy Explorer

Explore the interactive hierarchy to see how disciplines and subdisciplines connect. For example, discover how Computer Science links to Cognitive Psychology through AI Ethics.

🔹 Network Analysis

Our network analysis reveals the most central disciplines (e.g., Computer Science, Biology) and the types of connections (e.g., Topic-to-Concept, Subdiscipline-to-Topic).

Read our essays on interdisciplinary topics, including:
